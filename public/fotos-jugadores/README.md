# Fotos de jugadores (cumpleaños, perfiles)

## Formato del archivo

- **Extensión:** `.jpg` (recomendado) o `.png`
- **Tamaño:** 400×400 px o 520×390 px (ratio 4:3)
- **Peso:** idealmente &lt; 200 KB

## Nombre del archivo

`nombre-apellido-paterno.jpg` en minúsculas, sin tildes ni espacios.

**Ejemplos:**
- Felipe Huidobro García → `felipe-huidobro.jpg`
- Andrés Parker Parada → `andres-parker.jpg`
- Sebastián Vega Hidalgo → `sebastian-vega.jpg`

## Dónde se usan

- Tarjeta "El cumpleaños Maestro"
- Perfil del jugador en `/jugadores/[id]`
